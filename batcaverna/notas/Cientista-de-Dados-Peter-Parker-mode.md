# Checklist de Metas: Trilha Fullstack & Desktop (Python, JS, Rust)_x000d__x000d_
---_x000d__x000d_
_x000d__x000d_
Este checklist visa integrar as tecnologias mencionadas (Flask, Pandas, PySide6, NextJS, Tauri, Rust) em projetos práticos de complexidade crescente._x000d__x000d_
_x000d__x000d_
## Fase 1: Backend & Processamento de Dados (Python)_x000d__x000d_
_x000d__x000d_
Foco: Flask, Pandas, Modelagem de Dados, API REST._x000d__x000d_
_x000d__x000d_
- [ ] **Projeto 1: API de Análise de Dados (Simples)**_x000d__x000d_
  - *Stack:* Flask, Pandas._x000d__x000d_
  - *Objetivo:* Criar uma rota POST que recebe um arquivo CSV/JSON, processa estatísticas básicas (média, mediana, desvio padrão) com Pandas e retorna um JSON com os resultados._x000d__x000d_
- [ ] **Projeto 2: Sistema de Gestão de Inventário (CRUD REST)**_x000d__x000d_
  - *Stack:* Flask, SQLAlchemy (ou outro ORM), SQLite/Postgres._x000d__x000d_
  - *Objetivo:* Criar uma API REST completa com autenticação básica. Rotas para criar, ler, atualizar e deletar produtos._x000d__x000d_
  - *Meta:* Implementar validação de dados (Pydantic ou Marshmallow)._x000d__x000d_
_x000d__x000d_
## Fase 2: Desktop Nativo & GUI (Python)_x000d__x000d_
_x000d__x000d_
Foco: PySide6 (Qt for Python)._x000d__x000d_
_x000d__x000d_
- [ ] **Projeto 3: Dashboard de Monitoramento de Sistema**_x000d__x000d_
  - *Stack:* PySide6, psutil._x000d__x000d_
  - *Objetivo:* Criar uma janela desktop que mostra uso de CPU, RAM e Disco em tempo real usando gráficos (PyQtGraph ou Matplotlib integrado)._x000d__x000d_
- [ ] **Projeto 4: Cliente Desktop para Inventário**_x000d__x000d_
  - *Stack:* PySide6, requests._x000d__x000d_
  - *Objetivo:* Criar uma interface gráfica que consome a API do *Projeto 2*. O usuário deve poder adicionar e ver produtos através do app desktop._x000d__x000d_
